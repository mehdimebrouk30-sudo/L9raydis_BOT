## ⚙️ الميزات

- **منشن جميع أعضاء المجموعة** باستخدام الأمر `.tagall`
- **استخدام مقيد بالمشرفين** (فقط مشرفو المجموعة يمكنهم استخدام بعض الأوامر)
- **ألعاب** مثل إكس-أو (Tic-Tac-Toe) لتفاعل المجموعة
- **تحويل النص إلى كلام** باستخدام `.tts`
- **إنشاء الملصقات** باستخدام `.sticker`
- **كشف الروابط** لسلامة المجموعة
- **تحذير وإدارة الأعضاء** مع تحكم المشرفين

## 📖 عن البوت

بوت واتساب jawad.bot يساعد مشرفي المجموعات من خلال تزويدهم بأدوات لإدارة المجموعات الكبيرة بكفاءة. يستخدم البوت مكتبة Baileys للتفاعل مع واجهة برمجة تطبيقات واتساب ويب ويدعم ميزات الأجهزة المتعددة.

إنه خفيف الوزن ويمكن تخصيصه بسهولة لإضافة المزيد من الأوامر حسب متطلباتك. يعمل البوت في بيئة Node.js ويوفر مصادقة عبر رمز QR لربط حساب واتساب الخاص بك.

## 🛠️ الإعداد والتثبيت

### المتطلبات الأساسية

- Node.js مثبت على نظامك
- Git مثبت (لنسخ المستودع)

### التثبيت خطوة بخطوة

1. **انسخ المستودع:**

    ```bash
    git clone https://github.com/mruniquehacker/Knightbot-MD.git
    cd Knightbot-MD
    ```

2. **قم بتثبيت التبعيات:**

    ```bash
    npm install
    ```

3. **شغّل البوت:**

    ```bash
    node index.js
    ```

4. **امسح رمز QR:**

    بمجرد بدء تشغيل البوت، سيظهر رمز QR في الطرفية. امسح رمز QR هذا باستخدام ميزة الأجهزة المرتبطة في واتساب لربط حساب واتساب الخاص بك مع البوت.
