# 🍁 أوامر المشرفين

*╭━━━━━━━━🍁━━━━━━━━╮*

*┃اوامر المشرفين*

*┃━━━━━━━━━━━━━━━━━━┃*

🚫 *.kick* `طرد`

🚮 *.deletel* `حذف`

⚠️ *.warn* `تحذير`

📵 *.anticall off* `منع المكالمات`

☎️ *.anticall on* `تفعيل المكالمات`

🖇️ *.antilink off* `سماح بالروابط`

📛 *.antilink on* `منع الروابط`

♻️ *.antitag off* `تفعيل طاغ الكل`

〽️ *.antitag on* `تعطيل طاغ الكل`

👋 *.welcome on* `تفعيل ترحيب`

👊 *.welcome off* `تعطيل ترحيب`

🚯 *.ban* `منع استخدام البوت`

❇️ *.unban* `تفعيل استخدام البوت`

# 🚀 تشغيل البوت على Katabump

### 1️⃣ إنشاء سيرفر

قم بإنشاء حساب على Katabump ثم أنشئ سيرفر Node.js جديد.

### 2️⃣ استنساخ المستودع

```bash
git clone https://github.com/darkxecutor/JAWAD_BOT.git
cd JAWAD_BOT
```

### 3️⃣ تثبيت الحزم

```bash
npm install
```

### 4️⃣ تشغيل البوت

```bash
npm start
```

أو

```bash
node .
```
