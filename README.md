📱 التشغيل على Termux (أندرويد)

```bash
# تحديث الحزم
pkg update && pkg upgrade -y

# تثبيت المتطلبات
pkg install git nodejs ffmpeg -y

# تحميل البوت
git clone https://github.com/DarkXecutor/JAWAD.BOT.git

# الدخول إلى المجلد
cd JAWAD.BOT

# تثبيت الاعتماديات
npm install

# تشغيل البوت
npm start
```

💻 متطلبات التشغيل الأساسية

المتطلب الإصدار ملاحظات
Node.js v18+ مطلوب لتشغيل JavaScript
ffmpeg أحدث لمعالجة الصوت والفيديو
RAM 512MB+ الحد الأدنى
التخزين 1GB+ مساحة كافية

⏰ ملاحظة: البوت مبني بلغة JavaScript ويستغرق إعداده بالكامل حوالي 3 أشهر من العمل المتواصل.
