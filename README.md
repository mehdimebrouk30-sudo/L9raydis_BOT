## 🚀 طرق تشغيل البوت

### الطريقة الأولى: التشغيل على سيرفر katabump.com (مستضاف 24 ساعة)

1. **إنشاء حساب** على [katabump.com](https://katabump.com).
2. **إنشاء خدمة جديدة** (New Service).
3. اختر `Git Clone` وأدخل رابط المستودع:
```

https://github.com/darkxecutor/JAWAD_BOT.git

```
4. في إعدادات `Start Command` اكتب:
   ```bash
   npm start
```

5. أضف المتغيرات البيئية التالية (إن وجدت في البوت) مثل SESSION_ID إذا كان مطلوبًا.
6. اضغط على Deploy وانتظر حتى يتم التثبيت والتشغيل.
7. امسح رمز QR الذي يظهر في سجل السيرفر عبر تطبيق WhatsApp (الإعدادات > الأجهزة المرتبطة).

✅ ملاحظة: السيرفر يعمل 24/7 ولكن قد يتوقف إذا كان الخطة مجانية. يفضل استخدام خطة مدفوعة لضمان الاستمرارية.

---

الطريقة الثانية: التشغيل على Termux (Android)

1. فتح Termux وتحديث الحزم:
   ```bash
   pkg update && pkg upgrade -y
   ```
2. تثبيت Git و Node.js:
   ```bash
   pkg install git nodejs-lts -y
   ```
3. استنساخ المستودع:
   ```bash
   git clone https://github.com/darkxecutor/JAWAD_BOT.git
   cd JAWAD_BOT
   ```
4. تثبيت الاعتماديات:
   ```bash
   npm install
   ```
5. تشغيل البوت:
   ```bash
   npm start
   ```
6. مسح رمز QR عند ظهوره عبر WhatsApp (الإعدادات > الأجهزة المرتبطة).

💡 نصيحة: لمنع انقطاع البوت عند قفل الشاشة، يمكنك استخدام تطبيق Termux:Boot أو تشغيله داخل tmux أو screen.
