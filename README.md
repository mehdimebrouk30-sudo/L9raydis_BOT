# 🍁 أوامر المشرفين

╮━━━━━━━━🍁━━━━━━━━╭

┃ **اوامر المشرفين**

┃━━━━━━━━━━━━━━━━━━┃

🚫 **.kick** `طرد عضو`

🚮 **.deletel** `حذف رسالة`

⚠️ **.warn** `تحذير عضو`

📵 **.anticall off** `تعطيل منع المكالمات`

☎️ **.anticall on** `تفعيل منع المكالمات`

🖇️ **.antilink off** `السماح بالروابط`

📛 **.antilink on** `منع الروابط`

♻️ **.antitag off** `السماح بمنشن الكل`

〽️ **.antitag on** `تعطيل منشن الكل`

👋 **.welcome on** `تفعيل الترحيب`

👊 **.welcome off** `تعطيل الترحيب`

🚯 **.ban** `منع استخدام البوت`

❇️ **.unban** `السماح باستخدام البوت`

# 🚀 تشغيل البوت على Katabump

## 1️⃣ إنشاء سيرفر

* قم بإنشاء حساب في Katabump.
* أنشئ سيرفر Node.js جديد.

## 2️⃣ استنساخ المستودع

```bash
git clone https://github.com/darkxecutor/JAWAD_BOT.git
cd JAWAD_BOT
```

## 3️⃣ تثبيت الحزم

```bash
npm install
```

## 4️⃣ تشغيل البوت

```bash
npm start
```

أو

```bash
node .
```
