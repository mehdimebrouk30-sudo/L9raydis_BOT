## ✨ المميزات

- 🛡️ نظام حماية متطور للمجموعات
- 🔗 أمر `antilink on` لحذف الروابط تلقائياً
- 👥 إدارة الأعضاء بسهولة
- 🚫 منع السبام والإزعاج
- ⚡ سرعة عالية في تنفيذ الأوامر
- 🤖 أوامر ترفيهية وإدارية متنوعة
- 🔒 حماية متقدمة للمشرفين

---

## 📋 الأوامر الأساسية

| الأمر | الوصف |
|---------|---------|
| `.menu` | عرض قائمة الأوامر |
| `.antilink on` | تفعيل حذف الروابط |
| `.antilink off` | إيقاف حذف الروابط |
| `.tagall` | منشن لجميع أعضاء المجموعة |
| `.kick` | طرد عضو |
| `.promote` | ترقية عضو إلى مشرف |
| `.demote` | إزالة الإشراف |
